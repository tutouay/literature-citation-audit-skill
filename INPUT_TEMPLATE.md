# Literature Citation Audit Input Template

Complete the following form.

```text
Output language:

Citation style:
Choose APA 7, Chicago, Harvard, IEEE, Vancouver, a journal-specific style, a custom style, or "metadata check only."

Target paragraph, passage, or section:

Optional reference information:
Add titles, DOI values, links, or complete entries when available. Leave blank when unavailable.

Optional source passage you intended to cite:
Leave blank when unavailable.
```

A passage containing in-text citations triggers a full audit by default.

Do not upload PDFs for the initial audit. The AI should search first and request a PDF only when full text cannot be accessed or verified.


When complete reference entries are absent, the AI should reconstruct the bibliographic information in the first table column and include a DOI link or an official link to the corresponding work.
