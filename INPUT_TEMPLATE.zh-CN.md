# 文献引用核查输入模板

请直接填写以下内容。

```text
输出语言：

参考文献格式：
可填写 APA 7、Chicago、Harvard、IEEE、Vancouver、目标期刊格式、自定义格式，或“只核对文献信息，不检查格式”。

待核查段落、连续文本或章节：

可选参考文献信息：
有则填写题名、DOI、链接或完整条目。没有可以留空。

我原本想引用的文献原句或段落：
有则填写，没有可以留空。
```

提交带文内引用的文本后，默认完整核查全部引文。

首次核查不要求上传 PDF。AI 应先自行检索，无法获取或核验全文时再请求 PDF。


用户未提供完整参考文献条目时，AI 应在核查表第一列补充完整书目信息和 DOI 链接，或提供可直接打开对应文献页面的正式链接。
